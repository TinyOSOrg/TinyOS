ext_variant文件系统在内存中的布局：
  ##################################################################
  #             #           #             #           #            #    
  #  superblock # inode_map # inode_table # block_map # data_block #
  #             #           #             #           #            #     
  ##################################################################
  其中的每一部分都占用整数个块，文件系统的第一个块总是超级块且超级块仅占用一个块的大小。
